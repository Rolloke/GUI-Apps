Supported Protocols

This document describes the protocols used by Axis video products for different purposes. 

Note: Supported Functionality and Protocols are highly product and firmware version dependent.

Functionality Protocols
Motion JPEG and audio streams HTTP
MPEG-4 and audio streams as multicast RTSP or SDP file for media announcement.
RTP/UDP for media transmission.
IGMP for multicast support through routers.
RTCP for RTP control.
MPEG-4 and audio streams as unicast RTSP for media announcement and control.
RTP/UDP, RTP/RTSP or RTP/RTSP/HTTP for media transmission.
RTCP for RTP control.
Event notifications SMTP
HTTP
TCP
Image upload SMTP
HTTP
FTP
Network control data DHCP
UPnP
ARP
DNS
dynDNS
SOCKS
SNMP
