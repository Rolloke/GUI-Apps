AXIS VAPIX™ API

Axis Communications has introduced AXIS VAPIX as a means to address the growing interest from network video development partners. AXIS VAPIX is considered to be the global application programming interface (API) standard, in the rapidly growing field of Network Video based on IP.

All Axis network cameras and video servers have an HTTP-based application programming interface. This AXIS VAPIX HTTP API provides functionality for requesting images, controlling network camera functions (PTZ, relays etc.) and setting/retrieving internal parameter values. The purpose of the API is to make it easier for developers to build applications that support Axis video products.
