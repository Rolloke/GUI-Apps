AXIS VAPIX™ API Extensions for IV Modules

IV Modules are special purpose software modules designed to run on a DSP (Digital Signal Processor). The first product to include a DSP is the AXIS 242S IV Video Server.

IV Modules will be provided both by Axis, and by some of Axis ADP Partners. They have the ability to extend the API of the product by new API calls, new parameters and can have data  channels to communicate with Host Applications.

The modules provided by Axis will be treated as modular extensions to the VAPIX API and all have Open API:s so that any 3rd party developer can integrate support for them.
