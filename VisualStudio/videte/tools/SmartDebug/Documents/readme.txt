Installation:
Setup.exe aufrufen.

Beschreibung:
SmartDebug ist ein VisualStudio Plugin.
Es zeigt den Inhalt eines allocierten Arrays in einem definierbaren Bereich (z.B.: 0 - 100) an.
Die Werte werden in eine Liste ausgegeben und können graphisch dargestellt oder als CSV in die Zwischenablage kopiert werden.
Der Graph hat zwar nur eine rudimentäre Achsenbeschriftung, aber die Kurvenwerte können durch einen Mausklick auf die Kurve im einem Editfenster (Value n : 1.1234) angezeigt werden.

Die Anzahl der Werte sollte nicht zu groß gewählt werden. Bei 10000 Werten wartet man schon ein paar Sekunden.
