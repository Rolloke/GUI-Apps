Compact Flash Laufwerke in Windows

Compact Flash Laufwerke werden meistens als Removeable Media eingebunden. Dies können nicht partitioniert werden. Das Betriebssystem erkennt auch keine weiteren Partitionen, die mit Hilfe von fdisk o.ä. Programmen erzeugt wurden. Wenn man das Flag für removeable ändert, so können auch Compact Flash Geräte mehrere Partitionen haben.

Removeable Die Images können komplett z.B.: auf einer Festplatte erstellt werden und müssen nur noch kopiert werden. Der FBA muss nicht auf dem Compact Flash Gerät laufen. Können nicht partitioniert werden. Es werden auch keine weiteren Partitionen erkannt. Diese währen für den EWF notwendig, um die Einstellungen für den EWF zu speichern.
Ein komplett erstelltes Image kann schon vorher mit allen Einstellungen getestet werden. Nur EWF RAM-REG Modus möglich. Dieser muss nachträglich über Registryeinträge auf der CF Karte aktiviert werden.
Nachträgliche Komprimierung der Betriebssystemkomponenten. auf dem Compact Flash ist möglich. (256 MB CF Karte) Daten können nur auf einer zusätzlichen Festplatte permanent gespeichert werden.
Es ist nur eine Partition notwendig. Permanente Speicherung auf dem CF nur durch Reboot möglich.
Herkömmliche CFs können verwendet werden.  
Fixed Können partitioniert werden. Auf weiteren Partitionen können Daten für den EWF und auch für Anwendungen gespeichert werden. Für die Erstellung der EWF Partition für den RAM Modus muss der FBA auf dem Compact Flash laufen.
Auf einer ungeschützten Partition auf dem CF  können die Anwendungsdaten kontrolliert permanent gespeichert werden. Die Einstellungen für das Betriebssystem müssten nachträglich vorgenommen werden.
