		- Weitere Prozesse, die im System gestartet sind und für den reibungslosen Betrieb 
		  notwendig sind, wurden zu den Defaulteinträgen in der Registry hinzugefügt:
			- services.exe = Ignore: Systemservices for WinXP Plug and Play
			- spoolsv.exe  = Ignore: Printerspooler 
			- svhost.exe   = Ignore: Generic Host Process for Win32 Services
			- lsass.exe    = Ignore: The Lsass.exe process is responsible for management of
			  local security authority domain authentication and Active Directory management.
			  This process handles authentication for both the client and the server, and
			  it also governs the Active Directory engine. The Lsass.exe process is
			  responsible for the following components:
				- Local Security Authority
				- Net Logon service
				- Security Accounts Manager service
				- LSA Server service
				- Secure Sockets Layer (SSL)
				- Kerberos v5 authentication protocol
				- NTLM authentication protocol
			- winlogon.exe = Ignore: The Windows NT security subsystem includes the
			  implementation of secure interactive login support, which is
			  controlled by winlogon.exe. Winlogin.exe gets executed early in
			  the process and ensures that the user provides the proper
			  credentials (proof) before obtaining access to the computer.
			- csrss.exe    =  Ignore: Csrss stands for client/server run-time subsystem and is
			  an essential subsystem that must be running at all times. Csrss is
			  responsible for console windows, creating and/or deleting threads,
			  and some parts of the 16-bit virtual MS-DOS environment
			- smss.exe     = Ignore: This is the session manager subsystem, which is responsible
			  for starting the user session. This process is initiated by the system thread and
			  is responsible for various activities, including launching the Winlogon and Win32
			  (Csrss.exe) processes and setting system variables. After it has launched these
			  processes, it waits for either Winlogon or Csrss to end. If this happens
			  "normally," the system shuts down. If it happens unexpectedly, Smss.exe causes
			  the system to stop responding (hang)."
			- rundll32.exe = Control:
			  Kontrolliert die Installation von Treibern
			- rundll32<Files Needed> als Key mit den folgenden Einträgen:
			  - Message01 = PASSWORD
			  - NotOk als Key mit:
			    - DlgItem   = 0x6e
				- Message01 = ENABLE
				- wParam01  = 0
			  - Ok als Key mit:
			    - DlgItem   = 0x6e
				- Message01 = ENABLE
				- wParam01  = 1
		- Bestehende Defaulteinträge die geändert wurden
			- DVHook.exe    = Control: Selektive kontrolle über die Fenster
				- DVHook.exe<DVHook> = Hide		: Das DVHook-Fenster wird vesteckt
				- DVHook.exe<*>		 = Ignore	: Alle anderen ignoriert
