LogProfiler Description

The logProfiler synchronizes log file entries with various time stamp formats.
Retrieve the time of a log file line with the button 'Get Time', the time is shown within the time controls below the buttons.
Open another log file and use the button 'Find time' to locate the line with the nearest time stamp.

If the time format of the log file lines differs from the supported time formats, you can add your own time format to the list.
Set the text cursor in a line with a time entry and edit the format with the edit control 'Test Time Format'.
Take the list entries as example and use the specifiers in the table below to create your desired time format and test it with 'Get Time'.
The time entry of should be shown in the 'Date' and 'Time' controls, if the format is correct.
Insert the correct time format into the list with the button 'Insert'. Remove a selected entry from the list with the button 'Delete'.
If the time format does not contain a date, the date in the according 'Date' control is assumed for the search.
